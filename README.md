## Project Overview

This project focuses on **Linux Process Scheduling**, a core component of the Linux operating system.

It explores how Linux manages multiple processes efficiently using scheduling algorithms such as:

* First Come First Serve (FCFS)
* Round Robin
* Completely Fair Scheduler (CFS)

The goal is to understand how fairness and performance are balanced in a multitasking system.
